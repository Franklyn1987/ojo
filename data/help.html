Ojo is best controlled via the keyboard. Here are the shortcuts you need to know:

Browse and image modes

F11- fullscreen mode
Alt-F4- exit Ojo

Image mode

Enter - switch to browse mode, i.e. show all images in current folder and allow navigating the folder tree
Arrowsand PgUp/PgDown- move back and forth between images
Home/ End- go to first/last image
Press and hold mouse to zoom to specific point at 100%, then hold and move to "look around"
Z- toggles zoom between 100% and Fit-to-window (partial zooming is not supported yet)
Esc- return to browse mode if you were there before, or exit if you just opened an image

Browse mode

Enter - open selected image or folder
Arrowsand PgUp/PgDown- navigate around
Home/ End- go to first/last image or folder
Tab/ Shift-Tab- switch focus between files and folders list
Backspace/ Alt-Up/ Ctrl-Up- move one folder up
Ctrl-/- jump to the filesystem root
Alt-Left/ Alt-Right- go back / go forward in history
Ctrl +/=/ Ctrl -- increase / decrease size of the thumbnails
F5- reload current folder (file changes are not reflected automatically)
Ctrl-F5- refresh/recreate all thumbnails in the current folder
Ctrl-For type text directly- enter search/filter mode. Use this to filter images, folders and commands. E.g. type .jpgto see just JPEG files. Or type dateto focus the Sort by datecommand. The filtered view over the images remains if you go into Image mode and start cycling the images. Press Escto clear the filter and exit search mode (but keep in mind a second Escwill exit Ojo).
